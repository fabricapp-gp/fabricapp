Rapid Build Guide

The platform is optimized for seamless deployment on Vercel or local industrial servers.

Terminal Commands

# 1. Start Backend (API Metrics Engine)
cd api-server
python main.py

# 2. Start Frontend (Live Control Tower)
npm run dev
Prerequisites
  • • Node.js 18+ & Python 3.10
  • • Firebase Admin Access
  • • SMTP App Password
Critical Tip

Ensure .env.local contains your Firebase config to maintain data persistence across restarts.

Interactive Click-Map

Follow this exact path to navigate the app effortlessly from a cold start.

Stage 1

Initializing the Brain

Click "Forecast Engine" in the sidebar. This is where the intelligent mapping begins.

1. Click "Upload Sales CSV" Select your raw order file from your local drive.
2. Click "Run Forecast" Watch the system clean data and map families (Bianca, Onyx, etc).
Stage 2

The Strategic Overview

Navigate to the "Dashboard". This is your command center for risk analysis.

1. Adjust Inline Quantities Click any Inventory or Lead Time box and type a new number.
2. Manual Email Alert Click the Mail Icon near the Save button to send targeted vendor alerts.
Stage 3

Recipe Management

Open the "Studio". Here you define the physical properties of your garments.

1. "Add New Style BOM" Enter Style Name and fabric consumption (now explicitly in meters).
2. Manage Lifecycle Toggle the Archive icon to hide old styles or the Trash icon to delete.
Stage 4

Security & Governance

Access the "Admin Panel". Oversight for the entire operation.

1. Manage Users Add new team members and assign "Admin" vs "Viewer" roles.
2. Audit Trail Scroll through the log to see exactly who performed what action and when.
